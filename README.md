# DCF
Lecture on Discounted Cash Flows

On this website, I have compiled materials for the book Stochastic Discounted Cash Flow by Lutz Kruschwitz and me. The book itself is open access and can be downloaded from https://link.springer.com/book/10.1007/978-3-030-37081-7. This site also links to videos accompanying the book and includes a syllabus with additional exercises and model solutions. I used these materials to teach a 3-hour per week course over 14 weeks.

Die gesamte Videolaufzeit von „Basic Elements“ beträgt etwa 1:50h, die gesamte Videolaufzeit von "Corporate Income Tax" beträgt etwa 2:50h.
